# Lizard TUI

Terminal UI for visualizing Lizard code complexity analysis.

## Overview

A terminal-based user interface that wraps the Lizard code complexity analyzer, providing a visual and interactive way to explore cyclomatic complexity metrics for your codebase.

## Usage

```bash
pip install .
python lizard_tui.py
```

## Requirements

- Python 3
- Lizard (code complexity analyzer)
- See `pyproject.toml` for full dependencies
